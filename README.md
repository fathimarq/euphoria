# Euphoria

Euphoria is a pixel-art habit tracker where every good thing you do grows a plant.

## Features

1. Log one good thing daily and watch it grow from seed to legendary plant
2. Collect plant cards with motivational quotes on the back
3. Build your own garden over time
4. Share wins in The Meadow community wall
5. Add friends, visit their gardens, and send messages
6. Level up from Seedling to Euphoria Legend
7. Pick your own pixel avatar

## Built With

HTML5
CSS3
Vanilla JavaScript
LocalStorage

## Try It Live

https://fathimarq.github.io/euphoria/

## Run Locally

1. Download or clone this repository
2. Open index.html in your browser
3. Sign up and start planting

## About

Most habit trackers punish you for missing days. Euphoria does the opposite - it celebrates everything you do. Every plant in your garden is proof that you showed up for yourself.
